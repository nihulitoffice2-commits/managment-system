# Management System - ניהולית

מערכת ניהול פרויקטים ומשימות מקיפה לעמותות וארגונים.

## Features

- ✅ ניהול פרויקטים ומשימות
- ✅ לוח קנבן אינטראקטיבי
- ✅ לוח שנה ופגישות
- ✅ גאנט ניהולי
- ✅ דוחות ונתונים
- ✅ ניהול משתמשים והרשאות
- ✅ אינטגרציה עם Firebase

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
